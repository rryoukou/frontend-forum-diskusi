🐛 TROUBLESHOOTING

Issue: Tests fail with timeout
Fix: Ensure backend & frontend are running

Issue: Login fails
Fix: Check credentials in cypress/fixtures/admin.json
     Verify backend is seeded: php artisan db:seed

Issue: Elements not found
Fix: Add cy.wait(500) before interactions
     Use Cypress Selector Playground

Issue: Flaky tests
Fix: Use cy.intercept() instead of cy.wait()
     Add explicit visibility checks
